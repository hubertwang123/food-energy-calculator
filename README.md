# 食物能量计算器

一个简单、实用的网页应用，帮助你快速记录和计算每日饮食的能量和营养成分。

## 功能特点

### 核心功能

- **快速搜索食物**：内置45+种常见食物数据库，支持实时搜索和分类筛选
- **每日记录**：记录每天摄入的食物，自动计算总热量、蛋白质、脂肪和碳水化合物
- **自定义食物**：添加自己的食物数据，满足个性化需求
- **数据导出**：一键导出饮食记录为CSV格式，方便进一步分析
- **历史记录**：查看过往30天的饮食记录和营养摘要

### 技术特性

- **纯前端**：使用HTML/CSS/JavaScript开发，无需服务器
- **本地存储**：数据保存在浏览器LocalStorage，隐私安全
- **响应式设计**：完美支持桌面和移动设备
- **现代化UI**：简洁美观的界面设计

## 快速开始

### 安装

1. 下载或克隆此项目到本地
2. 直接用浏览器打开 `index.html` 文件即可使用

```bash
# 如果使用Git克隆
git clone <repository-url>
cd food-energy-calculator

# 在浏览器中打开
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

### 使用说明

#### 1. 添加食物到今日记录

- 在"今日记录"页面，使用搜索框查找食物
- 可以按分类筛选（主食、肉类、蔬菜、水果等）
- 点击"添加"按钮将食物加入今日记录
- 可以调整食物份量（支持小数）

#### 2. 查看营养摘要

- 页面顶部实时显示今日总计：
  - 总热量（千卡）
  - 蛋白质（克）
  - 脂肪（克）
  - 碳水化合物（克）

#### 3. 管理记录

- 调整份量：修改数量输入框后会自动更新营养数据
- 删除记录：点击"删除"按钮移除不需要的食物

#### 4. 添加自定义食物

- 切换到"自定义食物"页面
- 点击"添加自定义食物"按钮
- 填写食物信息：
  - 名称（必填）
  - 热量、蛋白质、脂肪、碳水化合物（必填）
  - 分类（可选）
- 自定义食物会出现在食物搜索列表中

#### 5. 查看历史记录

- 切换到"历史记录"页面
- 查看过往30天的饮食记录
- 每天显示详细的食物列表和营养总计
- 可以删除某天的所有记录

#### 6. 导出数据

- 点击底部"导出数据 (CSV)"按钮
- 自动下载包含所有历史记录的CSV文件
- 可用Excel、Google Sheets等工具打开

## 内置食物数据库

项目包含45+种常见食物，涵盖以下分类：

- **主食**：米饭、面条、馒头、包子、炒饭、水饺等
- **肉类**：鸡胸肉、猪肉、牛肉、鱼肉、鸡蛋、虾仁等
- **蔬菜**：西兰花、番茄、黄瓜、生菜、胡萝卜、土豆等
- **水果**：苹果、香蕉、橙子、葡萄、西瓜、草莓等
- **快餐**：汉堡、披萨、炸鸡、薯条、三明治等
- **零食饮料**：薯片、饼干、巧克力、可乐、奶茶、牛奶等
- **豆制品**：豆腐、豆浆、豆干等
- **坚果**：核桃、杏仁、腰果等
- **其他**：调料、乳制品、甜品等

## 数据存储

- 所有数据保存在浏览器的LocalStorage中
- 数据仅存储在本地，不会上传到服务器
- 清除浏览器数据会删除所有记录
- 建议定期导出数据备份

## 注意事项

1. **数据准确性**：内置食物数据为参考值，实际营养成分可能因品牌、做法而异
2. **份量单位**：大部分食物已注明重量，请根据实际情况调整份量
3. **数据备份**：定期使用"导出数据"功能备份记录
4. **隐私保护**：所有数据仅保存在本地浏览器，不会上传

## 常见问题

**Q: 数据会丢失吗？**
A: 数据保存在浏览器LocalStorage中，只要不清除浏览器数据就会一直保留。建议定期导出备份。

**Q: 可以在手机上使用吗？**
A: 可以！网站采用响应式设计，在手机上也能正常使用。

**Q: 如何添加更多食物？**
A: 可以使用"自定义食物"功能添加自己的食物数据。

**Q: 导出的CSV文件如何查看？**
A: 可以用Excel、Google Sheets、Numbers等表格软件打开。

**Q: 可以修改已有食物的数据吗？**
A: 内置食物数据无法修改，但可以创建自定义食物替代。
